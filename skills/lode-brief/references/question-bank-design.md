# 设计问题库（Design）— 起步模板

> 由 lode-brief 使用。读写都在本文件（`skills/lode-brief/references/question-bank-design.md`）——自进化只往这里增改，不另存 `.lode/` 副本。
> 把"感觉"敲定成具体决策：选择题优先，必问反参考，被否就换一批。
> 自进化只往这里增/改/删，每条对应一次真实失败。

| 维度 | 问题（选择题式 + 给推荐） | ✅ 能接受 | ❌ 打回去 |
|---|---|---|---|
| 设计方向 | ①编辑杂志 ②新拟态 ③玻璃拟态 ④Swiss…（建议 X） | 明确选一种 | "clean minimal"之类空话 |
| 组合参考 | 「A 的结构 + B 的气质」给 2–3 组挑 | 选一组 + 说哪里像 | 说不出参照 |
| **反参考** | **绝对不能出现什么？讨厌哪些产品的感觉？** | 列出 1–3 个反例 | 留空（必问） |
| 主题色 | 深色 / 浅色 / 都做？ | 选一 | 模糊 |
| 配色 | 给一组语义色（被否就换一批） | 用户认可一组 | 卡在"太 AI 了"不换 |
| 信息密度 | 紧凑 / 中等 / 宽松？ | 选一 | —— |
| 字体 | 2–3 套搭配挑一 | 选一 | —— |
| 圆角层级 | 锐利工具 / 柔和现代 / 大圆角消费？ | 选一 | —— |
| 关键状态 | hover/focus/active/空/载入/错误怎么呈现？ | 都有交代 | 漏状态 |
| 文案语气 | 工具型 / Agent 拟人 / 混合？ | 选一（对齐 Spec） | —— |

> 追加项目专属维度；反参考这一行**必须问**。
